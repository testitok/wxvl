## 🛠️ 5 分钟极速部署（Ubuntu22.04 一键脚本）  
```
# 1. 安装系统依赖apt update && apt install -y php php-{xml,gd,mysqli,dom,cli,zip,curl} unzip composer# 2. 获取代码git clone https://github.com/78778443/QingScan.gitcd QingScan/code && composer install# 3. 启动 Web 服务php think run -p 80                  # 默认 80 端口，浏览器访问 127.0.0.1# 4. 导入数据库（新建 qingscan 库并执行 deploy/qingscan.sql）# 5. 启动调度脚本（后台常驻）chmod +x script.sh && ./script.sh   # 自动拉起所有扫描器进程
```  
## 🔧 扫描器批量安装（PHP 一键式）  
```
# 查看可安装列表php think install list# 一键全部（约 20+ 工具，含 Nmap、AWVS、Xray、Nuclei、Rad 等）php think install all# 按需安装php think install nmapphp think install awvsphp think install nuclei
```  
## 📺 使用流程（3 步走）  
1. 添加目标 支持单 IP、CIDR、域名、URL 文件上传，可设置端口范围、扫描深度、POC 集合。  
  
1. 一键扫描 平台自动分配任务队列 → 多引擎并发 → 实时进度条 → 漏洞数红点提醒。  
  
1. 结果导出 支持 HTML / Excel / PDF 报告，含漏洞证明、修复建议、复现请求，渗透测试交付即用。
