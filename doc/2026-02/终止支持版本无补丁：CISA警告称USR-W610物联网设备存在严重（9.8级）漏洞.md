#  终止支持版本无补丁：CISA警告称USR-W610物联网设备存在严重（9.8级）漏洞  
sec随谈
                    sec随谈  sec随谈   2026-02-24 00:59  
  
美国网络安全和基础设施安全局 (CISA) 发布警告，指出广泛部署的工业物联网设备存在多个严重漏洞。该警告重点指出济南优瑞物联网科技有限公司 (PUSR) 生产的 USR-W610 Wi-Fi 串口设备服务器存在一系列基础性安全缺陷。  
  
对于管理员来说，最令人担忧的消息可能是这些设备已经达到使用寿命终止 (EOL) 状态，这意味着近期不会有官方补丁来关闭这些数字时代的大门。  
  
该批次中最具破坏性的缺陷是 CVE-2026-25715，其严重性评级为 9.8。  
  
该漏洞源于设备管理设置中的一个基本逻辑错误。根据安全公告，“该设备的Web管理界面允许将管理员用户名和密码设置为空值”。  
  
一旦应用此配置，将对网络完整性造成灾难性后果。报告警告称，“该设备允许通过 Web 管理界面和 Telnet 服务使用空凭据进行身份验证”。这“实际上禁用了所有关键管理通道的身份验证，使任何网络相邻的攻击者无需凭据即可获得完全的管理控制权限”。  
  
除了登录绕过问题之外，USR-W610 还存在缺乏现代加密标准和用户界面安全性差的问题：  
- 明文拦截漏洞 (CVE-2026-24455)：该设备不支持 HTTPS/TLS，而是依赖过时的 HTTP 基本身份验证。这导致“流量被编码但未加密，使用户凭据暴露于同一网络上的攻击者的被动拦截之下”。  
- 明文密码可见性漏洞 (CVE-2026-26049)：该漏洞严重威胁物理安全，其网页界面“以明文输入字段的形式呈现密码”。这一疏忽使得管理员凭据“对任何有权访问用户界面的人都直接可见”，使他们容易受到“偷窥、屏幕截图或浏览器表单缓存”等攻击。  
- Wi-Fi 取消认证漏洞 (CVE-2026-26048)：由于路由器缺乏管理帧保护，因此也容易受到局部干扰。这使得攻击者可以广播伪造的帧，从而“造成未经授权的干扰并导致拒绝服务”。  
供应商济南优瑞物联网科技有限公司已正式声明“该产品已停止支持，且无计划发布补丁”。这意味着USR-W610（3.1.1.0及以下版本）将永久存在安全漏洞，可能遭受“身份验证被禁用、拒绝服务攻击或攻击者窃取有效用户凭证”等情况。  
  
由于没有修复措施，CISA 和安全研究人员敦促用户立即停用受影响的设备，或将其隔离在严格划分的网络中，以防止未经授权的“网络邻近”参与者访问管理界面。
