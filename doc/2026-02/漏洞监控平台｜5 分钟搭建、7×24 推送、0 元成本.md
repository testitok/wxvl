## ⚙️ 5 步部署（CentOS 7 示例）  
### 装依赖  
```
pip3 install -r requirements.txt
```  
### 初始化数据库  
```
python3 Sql/installDb.py      # 自动生成 sqlite + log 目录
```  
### 配 Nginx（可选，Web 表格展示）  
```
yum install -y nginx# 修改 /etc/nginx/nginx.conflocation / {     root /usr/share/nginx/html/download;     autoindex on;           # 开启目录浏览     autoindex_localtime on;}
```  
### 配企业微信（10 分钟）  
```
创建企业 → 自建应用 → 拿到 CorpID + Secret填写到 config.ini 对应字段；自用请注释 toparty，取消 touser 注释
```  
### 启动监控  
```
nohup python3 monitor.py &
```
