## 📖 使用指南  
1. 管理 POC 模板 ① 侧边栏 → POC 模板 → 新建 ② 填写 id / name / severity → 在 Monaco 里写 YAML ③ 保存后自动高亮语法错误，一键推入「模板仓库」  
  
1. 执行扫描 ① 扫描器 → 输入目标（每行一个 URL/IP） ② 勾选 POC / 文件夹 → 并发线程数可调 ③ 启动 → 实时滚动日志 → 支持中途暂停  
  
1. 查看结果 ① 左栏按 critical / high / medium / low 聚合 ② 点击条目 → 右侧弹出 Request / Response 详情 ③ 支持「复制原始报文」「导出 CSV」  
  
1. 编码小工具 顶部菜单 → 工具 → 选择编码方式 → 实时转换，支持链式操作
