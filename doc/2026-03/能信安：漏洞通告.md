**Linuxkernel潜在死锁漏洞**  
  
![](https://mmbiz.qpic.cn/sz_mmbiz_png/7kDwzldia4nF5AC6icmiaUXuFePEEsH3L6OsJKOMeiczia5d6ZhKurrAbsOvxjgvOfxWpp2271BMNwAl9fMZjBHQ8QY1Ua90lw5WjA4evtqG9KoU/640?wx_fmt=png&from=appmsg "")  
  
  
  
**漏洞概况：**  
  
Linux kernel是美国Linux基金会的开源操作系统Linux所使用的内核。  
  
Linux kernel存在潜在死锁漏洞，该漏洞源于vgem-fence驱动中定时器释放未使用TIMER_IRQSAFE标志，攻击者可利用该漏洞导致拒绝服务。  
  
**受影响系统：**  
  
Linux Kernel ＜=6.17.12  
  
Linux Kernel  
  
**来源链接：**  
  
https://www.nsfocus.net/vulndb/134854  
  
**建议：**  
  
目前厂商已经发布了升级补丁以修复这个安全问题，请到厂商的主页下载：  
  
http://www.kernel.org/  
  
  
**0****2**  
  
**Apache Karaf Decanter不受信数据反序列化漏洞**  
  
![](https://mmbiz.qpic.cn/mmbiz_png/7kDwzldia4nF09K4n3IgrYHk861hRXRr9dX79pPYysHKuPwnF5tZPHPWf9pZb4caU9fn3W8Eygg5h5BhBeMoLKuWq466T4TQFQiaBnbjjENKE/640?wx_fmt=png&from=appmsg "")  
  
  
  
**漏洞概况：**  
  
Apache Karaf Decanter是Apache基金会的一个监控和告警模块。  
  
Apache Karaf Decanter 2.12.0之前版本存在不受信数据反序列化漏洞，该漏洞源于该模块在未认证的情况下开放了4560端口，其allowed classes属性配置可以被绕过，攻击者可利用该漏洞通过特制的不可信数据进行反序列化攻击，使目标用户系统出现拒绝服务。  
  
**受影响系统：**  
  
Apache Apache Karaf Decanter < 2.12.0  
  
**来源链接：**  
  
https://www.nsfocus.net/vulndb/134890  
  
**建议：**  
  
目前厂商已经发布了升级补丁以修复这个安全问题，请到厂商的主页下载：  
  
https://karaf.apache.org/  
   
  
  
**0****3**  
  
**HP Hotkey Support后置链接漏洞**  
  
![](https://mmbiz.qpic.cn/sz_mmbiz_png/7kDwzldia4nF1ibUHxVjRVuhFGhsx7PMnDPSQAoiabuEQRZ89zEGL1NP7YMg2R0Dbbibic05rHQUTHP0A6IWeU5LmNAxZsPibEk5hHM04PQiaJcTMY/640?wx_fmt=png&from=appmsg "")  
  
  
  
**漏洞概况：**  
  
HP Hotkey Support是美国惠普（HP）公司的一款应用程序。  
  
HP Hotkey Support的HPAudioAnalytics服务存在后置链接漏洞，攻击者可利用该漏洞提升权限。  
  
**受影响系统：**  
  
HP Hotkey Support  
  
**来源链接：**  
  
https://www.nsfocus.net/vulndb/134964  
  
**建议：**  
  
目前厂商已经发布了升级补丁以修复这个安全问题，请到厂商的主页下载：  
  
https://support.hp.com/us-en/document/ish_12893367-12893393-16/hpsbhf04044
