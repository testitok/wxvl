## 安装  
### 环境要求  
- Python 3.8+  
  
- Windows / Linux / macOS  
  
### 安装依赖  
```
pip install -r requirements.txt
```  
  
或者手动安装：  
```
pip install "httpx[http2,brotli]>=0.24.0"pip install "httpcore[asyncio]>=0.17.0"pip install mitmproxy>=10.0.0pip install PySide6>=6.4.0pip install beautifulsoup4>=4.11.0pip install Pillow>=9.0.0pip install lxml>=4.9.0
```  
### 运行程序  
```
python main.py
```  
  
或者双击运行 UploadRanger.bat  
 (Windows) 或 UploadRanger.sh  
 (Linux/macOS)
