#  台军空军关键设施被曝存在安全漏洞 紧急斥资2680万加固  
网空闲话
                    网空闲话  网空闲话plus   2026-03-07 00:14  
  
台自由时报3月6日报道，台当局内部将军事及政府关键设施视为所谓“安全韧性”的重要组成部分，台湾地区行政管理机构下设的“国土安全办公室”及相关安全部门会定期对军政关键设施进行安全防护指导与评估。台军政界人士6日透露，台军空军作战指挥系统部分关键设施被评估为安全防护不足，需要尽快补强。台军空军司令部本周已以“关键设施改善强化方案”为名，紧急委托岛内中山科学研究院（中科院）进行补强。  
  
台军空军作战指挥系统下辖13个一级单位，并设有一个任务编组——所谓“联合空中作战指挥中心”（JAOC）。此外还直辖“战管联队”“通信航管资讯联队”“气象联队”等三个作战与勤务支援单位。其主要职能是负责台海空中作战规划以及指挥管制，并督导执行相关战术任务，研究所谓“克敌战术战法”，运用自动化防空系统，对台军防空武器系统进行统一管制，以执行防空任务并支援其他军种作战，试图维持台海所谓“空中优势”，保障台湾及澎湖、金门、马祖等地区安全。  
  
为监控解放军军机活动并配合各阶段战备需求，台军作战指挥系统早在1968年就完成战管自动化防空系统建设，随后又于1994年8月12日启用战管“强网系统”承担战备任务。此后为配合台军所谓“第二代兵力整建”，按照既定计划分阶段扩充系统能力并提升功能，以强化对空情的掌控。

资料显示，台湾地区中山科学研究院在去年的防务航太展期间曾展示所谓“下一代智慧警监系统”，该系统整合可见光摄像机、红外摄像机及激光测距等多种感测设备。（图源：中山科学研究院）  
  
根据台湾地区防务主管部门本周公布的中标公告，台军空军司令部已在2月底委托中山科学研究院执行“关键设施改善强化方案”，项目金额约为2680万元新台币，并要求在2027年10月中旬之前完成。  
  
至于此次项目是否与中山科学研究院在台军各军事营区部署的智慧警监系统有关，还是涉及其他关键设施系统在安全防护方面存在漏洞与不足，台军方面未作进一步说明。
